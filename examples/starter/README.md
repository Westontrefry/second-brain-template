# Starter demo pack (U2)

Eight synthetic notes per PLAN-UX locked decision 7: six CS-flavored notes
wired to `dsa-interviews` (so readiness/gaps demo immediately) and two in
visibly non-CS domains (career, gen-ed) so the map shows domain-agnosticism
instead of claiming it. All carry `source: demo` and NO assessments —
locked decision 8: `brain demo --remove` must leave zero trace, and
assessment events are append-only.

Content authored + live-tested in the 2026-07-05 UX session (fresh-brain
sandbox simulation). Still TODO from the U2 milestone: the `brain demo
--install|--remove` command, and adding `demo` to config.yaml sources.

To demo ALL five evidenced levels (4–5 require quiz evidence, which this
pack deliberately omits): use the fresh-brain sandbox recipe in HANDOFF.md —
seed a BRAIN_ROOT sandbox with these notes, then `brain assess` a few of
them there through the sanctioned path. The sandbox is disposable, so the
no-orphaned-events rule is not violated.

The bodies deliberately model honest confidence judgment: level-1 notes say
things like "bookmarked material, not something I could code cold"; the
level-3 note shows a worked fix in the author's own words.
